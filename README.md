## 🚀 Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/mdzaheerjk/ML-Project-14-Credit-card-fraud-detection.git
   cd ML-Project-14-Credit-card-fraud-detection
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run analysis**
   ```bash
   python src/main.py
   ```
